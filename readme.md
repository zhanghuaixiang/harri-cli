## 创建项目
    harri create projectName

## 新增组件
    harri addcpn HelloWord
    或
    harri addcpn HelloWord -d src/components

## 新增模块
    harri addpage Index
    或
    harri addpage Index -d src/pages

## 指定目标目录
    -d 或 --dest
    
## 查看版本号
    harri --version 或 harri -V

## 查看指令帮助
    harri --help 或 harri -h
